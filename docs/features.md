## Features
  - **Basic Realtime Actions**: Start, Stop, and Restart services
  - **Advanced Realtime Actions**: Install and Delete services
  - **Security Auditing:** Automatic detection of *Unquoted Service Paths* and *Permissive Folder Permissions*.
  - **Quick Access to Popular Information:** View process uptime and the user running the service with zero extra clicks.
  - **Data Portability:** Export filtered views to CSV or copy the entire state as JSON to your clipboard.
  - **Modern UI:** Dark-themed, relatively responsive interface with a debounced search.
